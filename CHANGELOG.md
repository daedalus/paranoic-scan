# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

### Added
- Initial release
- Ported from Perl Paranoic Scan 1.7 (2014) to Python
- Core scanning modules (panel, SQLi, LFI, XSS, bypass)
- HTTP fingerprinting and port scanning
- Encoder/decoder utilities (Base64, Hex, URL, Binary, ASCII)
- MD5 encoding and cracking
- CLI with Click
